## Getting Started

Follow these steps to get started with the Elda Health Yoga website:

1. **Clone the Repository:**

  
   git clone https://github.com/Anshika31sharma/LeanBoard


2. **Install Dependencies:**

  
   npm install

3.**Start the Development Server:**

    npm run dev
